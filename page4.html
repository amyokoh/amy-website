Describe which are the appropriate filetypes for websites

PNG is a high quality format for images. It is lossless compression which means that it support high-quality images while retaining the original colour images and sharpness. The GIF file is a form of bitmap images, which means the graphic has many tiny parts called pixels. It is most suitable for storing graphics with a few colours, such as simple diagrams and shapes. JPG file format is based on lossy compression, meaning it losses it quality as you save. The compression ratio can be adjusted which means i can determine my own balance between storage size and quality.
